# Solace TPM Portfolio

A TPM portfolio case study built to demonstrate technical program management skills across tooling, dependency tracking, risk management, and stakeholder communication.

**Live site:** https://kat-perreira.github.io/solace-tpm-portfolio/

---

## About the project

Solace is a fictional B2B SaaS HR tech company. The case study documents a multi-workstream API integration program — Stripe for payroll disbursements and Workday for employee data sync — spanning five teams across three epics with a hard Q3 2026 launch deadline.

The portfolio covers how I structured the program, tracked dependencies and blockers, visualized risk, and documented the TPM decisions and tradeoffs made along the way.

---

## Tools used

- **Jira** — project structure, epics, stories, subtasks, blocker links, sprint planning
- **Tableau Public** — risk matrix and dependency status dashboard
- **GitHub Pages** — portfolio hosting

---

## What's inside

- Program timeline and epic structure
- Interactive risk register (12 risks scored by likelihood × impact)
- Dependency status dashboard (14 stories grouped by epic and status)
- Key Jira artifacts — board view, blocked ticket detail, vendor blocker ticket
- TPM decisions with reasoning and tradeoffs
- TPM retrospective — what went well, what I'd do differently, what comes next
